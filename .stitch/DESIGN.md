# Ketaflix Design System

## Brand Identity
- **App Name:** Ketaflix
- **Tagline:** Social movie tracking for friend groups
- **Logo:** Horse head silhouette SVG + "Ketaflix" wordmark with emerald gradient

## Platform
- Mobile-first PWA (390x844 viewport)
- Dark mode only

## Color Palette
| Role | Name | Hex | Usage |
|------|------|-----|-------|
| Background | Deep Black | #050608 | Page canvas, app background |
| Surface | Card Dark | rgba(255,255,255,0.04) | Card backgrounds, elevated surfaces |
| Primary Accent | Emerald | #34d399 | CTAs, active states, highlights, links |
| Secondary Accent | Warm Amber | #f59e0b | Star ratings, badges, secondary highlights |
| Primary Text | White | #ffffff | Headings, primary content |
| Secondary Text | White/55 | rgba(255,255,255,0.55) | Body copy, descriptions |
| Muted Text | White/35 | rgba(255,255,255,0.35) | Placeholders, tertiary info |
| Border | White/10 | rgba(255,255,255,0.10) | Card borders, dividers |

## Typography
- **Font Family:** Space Grotesk (Google Fonts)
- **Headings:** 600-700 weight, white
- **Body:** 400-500 weight, white/55
- **Captions:** 400 weight, white/35

## Visual Style
- **Cards:** Glassmorphism — semi-transparent bg, subtle white border, backdrop blur
- **Corners:** Generously rounded (12-16px)
- **Shadows:** Minimal — rely on glass effect and borders for depth
- **Icons:** Lucide icon set, 1.5px stroke weight
- **Posters:** 2:3 aspect ratio with rounded corners and rating badge overlay

## Logo
- **Horse Head:** UXWing silhouette SVG, emerald fill (#34d399), flipped horizontal — CC0 license
- **HorseIcon:** Horse-shaped rating icon (filled/dim states, like stars) — CC BY 4.0
- **Wordmark:** "Ketaflix" in Space Grotesk with emerald gradient

## Component Patterns
- **Bottom Nav:** 5 tabs — Feed (Clapperboard), Flix (Telescope), Krew (Sofa), Profile (UserRound), Settings (Settings) — all Lucide icons
- **Genre Pills:** Horizontal scroll, pill-shaped, subtle border, small icon prefix
- **Movie Poster Cards:** Glassmorphism frame, star rating badge (amber), title overlay
- **Avatar:** Circular with gradient background ring (emerald/amber)
- **Follow Button:** Outlined default, filled emerald on hover/active
- **Segmented Control:** Pill-shaped tabs with emerald active indicator

## Feature Names
- **Ketalogs** = Diary/Watched history (UI: Ketalogs)
- **Ketaqueue** = Watchlist/Queue (UI: Ketaqueue)
- **Ketacrew** = Rooms/Friend groups (UI: **Krew**)
- **Flix** = Discovery/Browse page (UI label for Discover tab)
